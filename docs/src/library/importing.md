# Importing Books

Cadmus scans your device's storage and adds books to its database automatically.
This process is called **importing**.

## Automatic import

Cadmus imports any new books it finds automatically. Copy files to your device,
and they'll appear in your library after the next import pass.

## Manual import

You can trigger an import at any time from the settings editor.

Cadmus will scan your library folder and add any books it hasn't seen before.
